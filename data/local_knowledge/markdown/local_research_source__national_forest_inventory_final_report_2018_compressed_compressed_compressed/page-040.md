       2.4              Data entry
       Before beginning with data entry phase, all the field forms were scanned for
       security reasons and the copies were stored in MEFCC hard drives. All
       collected data was then inputted into a database using Open Foris (OF) Collect
       software27. OF Collect is a user friendly, open-source and versatile software
       specifically designed for the data entry and cleansing, and its data can be
       exported into data analysis and result reporting. Several errors can be avoided
       by using this type of software, because the database contains a large se lection
       of lookup tables such as a predefined list of tree species names, toponyms,
       names of team members and validation rules for recorded attributes.
       Based on the designed survey the personnel responsible for the data entry can
       enter the data in an interface organized as the field form and directly populate
       a database. Because the software can be installed in a server, several experts
       can enter, modify and manage the field form data at the same time. By applying
       personal usernames and passwords, the system administrator can follow the
       progress of data entry. The backup system is automatic and all databases can be
       restored if necessary.
