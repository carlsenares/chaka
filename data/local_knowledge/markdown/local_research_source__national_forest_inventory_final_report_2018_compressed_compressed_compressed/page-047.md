2.8            DBH-Height models
Field conditions do not always allow field crews to successfully use the clinometer for
tree height measurement because the visibility of the top of tree that can sometimes be
partially or totally prevented by branches or other trees. For this reason, not all the
tree heights are measured in the field. In these cases tree height models that can be
applied to estimate the missing tree heights. Given that height is a fundamental factor
in estimating tree volume and biomass in the tropics37, it is important to ensure a
proper and consistent DBH/Height model to avoid losses on the results, mainly in the
REDD+ countries where the stock sequestration performances are a part of the
national economic strategy. Three different models, i.e. Naslund (1937), Schumacher
(1939) and Curtis (1967), were tested for the Ethiopia NFI dataset using the lmfor38
package in R, and by using only healthy living trees as input data. Curtis’ model (1967)
was ultimately selected since its better fit with the big trees in Forest Land:




h      =      estimated top height [m];

The parameters were estimated using nonlinear estimation techniques, and height
model parameters were computed by FRA classes by calibrating the model by sampling
