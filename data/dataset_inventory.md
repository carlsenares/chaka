## Dataset Details

### CHIRPS rainfall

- Dataset ID: `chirps_rainfall`
- Provider: Climate Hazards Center
- Coverage: near-global land
- Spatial resolution: 0.05 degree
- Temporal resolution: daily to monthly
- Access method: download_or_google_earth_engine
- Source: https://www.chc.ucsb.edu/data/chirps

Used features:

- rainfall_mean_mm
- rainfall_reliability_score

Known limitations:

- Resolution is coarse relative to 5-10km candidate sites.
- Do not over-weight rainfall compared with field hydrology.

### CIFOR-ICRAF Climate Change Atlas for Ethiopian FLR tree species

- Dataset ID: `cifor_icraf_species_atlas`
- Provider: CIFOR-ICRAF / World Agroforestry
- Coverage: Ethiopia
- Spatial resolution: species/model dependent
- Temporal resolution: atlas/model release
- Access method: web_atlas_or_reference
- Source: https://www.cifor-icraf.org/knowledge/publication/8977/

Used features:

- species_suitability_context
- species_suggestions

Known limitations:

- Use as context for suggested species, not as a deterministic field prescription.
- Local nursery availability and community preference still require validation.

### Global Forest Watch / UMD forest change layers

- Dataset ID: `gfw_umd_forest_change`
- Provider: Global Forest Watch / University of Maryland
- Coverage: global
- Spatial resolution: 30m
- Temporal resolution: annual loss layers with baseline tree cover
- Access method: download_or_api
- Source: https://data.globalforestwatch.org/

Used features:

- forest_loss_score
- tree_cover_context_score

Known limitations:

- Tree-cover loss is not the same as carbon loss or degradation severity.
- Threshold choices affect forest/non-forest interpretation.

- Spatial resolution: layer dependent
- Temporal resolution: multi-year releases

- Spatial resolution: 30m

- Access method: google_earth_engine

Used features:
